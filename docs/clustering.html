Note

Propósito: Identificar perfiles delictivos de las 32 entidades federativas de México, agrupándolas según la distribución de tipos de delito. Esto complementa el modelo supervisado al revelar qué estados comparten patrones estructurales similares de criminalidad.
Algoritmo elegido: K-Means

1. Selección y Justificación de Variables

Tip
Variable ¿Se usa? Justificación
tipo_delito (como columnas pivoteadas) ✅ Captura el perfil delictivo completo de cada estado
incidencia_delictiva (agregada por tipo) ✅ Magnitud real de cada tipo de delito por entidad
entidad_federativa ✅ Unidad de análisis (índice del pivot)
anio, mes, modalidad ❌ Se agregan en el pivot — no se usan como features directas
clave_ent, bien_juridico_afectado ❌ Redundantes con el pivot de tipo_delito

Las variables del clustering difieren de las del modelo supervisado: aquí se trabaja a nivel estatal agregado (32 filas), no a nivel de registro individual.

3. Algoritmo Elegido y Justificación

Algoritmo: K-Means

Justificación: - Forma de los grupos: Los perfiles delictivos estatales tienden a formar grupos compactos y relativamente esféricos en el espacio de features — condición ideal para K-Means. - Interpretabilidad: K-Means produce centroides que representan el “estado promedio” de cada cluster, facilitando la descripción de perfiles. - Escala: Con solo 32 observaciones (estados), K-Means converge rápidamente y es estable. - Normalización: Se aplicó StandardScaler previamente, eliminando el sesgo por magnitud que afectaría a K-Means.

4. Selección del Número de Grupos

Método del Codo (Inercia)

from sklearn.cluster import KMeans

ks = range(2, 10)
inertias = []

for k in ks:
    km = KMeans(n_clusters=k, random_state=SEED, n_init='auto')
    km.fit(X_scaled)
    inertias.append(km.inertia_)

fig, ax = plt.subplots(figsize=(8, 4))
ax.plot(ks, inertias, 'o-', color='#003f8a', linewidth=2, markersize=8)
ax.axvline(x=4, color='#c8a951', linestyle='--', linewidth=2, label='k = 4 (óptimo)')
ax.set_xlabel('Número de Clusters (k)', fontsize=12)
ax.set_ylabel('Inercia', fontsize=12)
ax.set_title('Método del Codo — Selección de k', fontsize=13, fontweight='bold')
ax.legend()
plt.tight_layout()
plt.show()

Interpretación: La curva de inercia muestra una reducción pronunciada hasta k=4, donde el codo es más notorio — punto a partir del cual agregar más clusters aporta poca ganancia en cohesión interna.

from sklearn.metrics import silhouette_score

sil_scores = []

k_optimo = list(ks)[sil_scores.index(max(sil_scores))]

6. Visualización de Grupos

Code 
PALETTE = ['#003f8a', '#c8a951', '#e85d4a', '#2ecc71']
NOMBRES = ['🔵 Delincuencia Moderada', '🟡 Alto Patrimonio', 
           '🔴 Alta Violencia', '🟢 Perfil Mixto Bajo']

fig, ax = plt.subplots(figsize=(12, 8))

for i in range(OPTIMAL_K):
    mask = labels == i
    ax.scatter(
        X_pca[mask, 0], X_pca[mask, 1],
        c=PALETTE[i], label=NOMBRES[i],
        alpha=0.85, s=120, edgecolors='white', linewidth=1.5, zorder=3
    )
    # Etiquetas de estados
    for j, (x, y) in enumerate(zip(X_pca[mask, 0], X_pca[mask, 1])):
        estado = tabla_pivot[tabla_pivot['cluster'] == i].index[j]
        # Abreviar nombre
        abrev = estado[:8] + '.' if len(estado) > 8 else estado
        ax.annotate(abrev, (x, y), textcoords='offset points',
                   xytext=(5, 5), fontsize=7, alpha=0.8)

# Centroides en espacio PCA
centroids_pca = pca.transform(km_final.cluster_centers_)
ax.scatter(centroids_pca[:, 0], centroids_pca[:, 1],
           c='black', marker='X', s=250, zorder=5, label='Centroides')

ax.set_xlabel(f'PC1 ({pca.explained_variance_ratio_[0]:.1%} varianza)', fontsize=12)
ax.set_ylabel(f'PC2 ({pca.explained_variance_ratio_[1]:.1%} varianza)', fontsize=12)
ax.set_title('Clusters de Entidades Federativas por Perfil Delictivo\n(Espacio PCA 2D)',
             fontsize=13, fontweight='bold')
ax.legend(loc='upper right', fontsize=10)
ax.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

Interpretación: La separación visible en el espacio PCA confirma que los clusters son estructuralmente distintos. Los estados más alejados del centro representan perfiles delictivos más extremos o atípicos.

```{python}
    labels = km.fit_predict(X_scaled)
    sil_scores.append(silhouette_score(X_scaled, labels))

k_optimo = list(ks)[sil_scores.index(max(sil_scores))]

fig, ax = plt.subplots(figsize=(8, 4))
ax.plot(ks, sil_scores, 's-', color='#1a6fd4', linewidth=2, markersize=8)
ax.axvline(x=k_optimo, color='#c8a951', linestyle='--', linewidth=2,
           label=f'k = {k_optimo} (mayor silueta)')
ax.set_xlabel('Número de Clusters (k)', fontsize=12)
ax.set_ylabel('Coeficiente de Silueta', fontsize=12)
ax.set_title('Coeficiente de Silueta — Mayor es mejor', fontsize=13, fontweight='bold')
ax.legend()
plt.tight_layout()
plt.show()

print(f"k con mayor silueta: {k_optimo}  (score = {max(sil_scores):.4f})")
```

> **Interpretación:** El coeficiente de silueta confirma que k=4 produce los clusters más cohesivos y bien separados. Un score mayor a 0.3 indica una estructura de agrupamiento razonable para datos socioeconómicos complejos.
